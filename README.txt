This is a pong game that runs from a python file.

The Pong.py will run when ran with a config file.  The config file contains elements such as ball radius, canvas size, paddle speed and things like that. PongClasses.py contains the classes that are used in the Pong.py file, classes include Paddle, and Ball.  These will be used as the main "hub" for all information related to the object.
